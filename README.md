# TanFit - Science-Based Nutrition Platform

TanFit is a comprehensive web application that provides personalized diet plans based on food science, sports science, anatomy, and biological sciences. The platform offers a beautiful green and blue theme that promotes comfort and health.

## 🌟 Features

### Main Website Features
- **Animated Background**: Machine-like moving background with floating elements
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Green & Blue Theme**: Calming colors that promote health and wellness
- **Modern UI/UX**: Beautiful animations and smooth interactions

### 🍽️ Diet Services
1. **Muscle Gain Diet** - Build lean muscle mass effectively
2. **Fat Loss Diet** - Lose fat while preserving muscle
3. **Happy Life Diet** - Optimal health and wellness for everyday living
4. **Professional Athletes** - Elite performance nutrition for competitive athletes

### 📊 BMI Calculator
- Interactive BMI calculation
- Health status categorization
- Personalized diet plan recommendations based on BMI results
- Real-time feedback and suggestions

### 🔐 User Authentication System
- **Sign Up**: Email and password registration with validation
- **Sign In**: Secure login with remember me option
- **Password Reset**: Email-based password recovery system
- **Email Validation**: Prevents duplicate accounts
- **Password Strength Indicator**: Real-time password security feedback

### 👤 User Profile Management
- **Profile Settings**: Update name, email, phone number
- **Password Management**: Change password securely
- **Order History**: View completed diet plans and purchases
- **Open Orders**: Continue incomplete questionnaires
- **User Statistics**: Track orders and membership duration
